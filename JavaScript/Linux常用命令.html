# Linux常用命令

# mkdir 创建文件夹

$ mkdir 文件夹名

-p 递归创建

$ mkdir -p cc/ee

# touch 创建文件

$ touch 文件名

$ touch cc/1.txt

# echo 写入内容（将hello写入到index.txt）

$ echo hello > index.txt
  • 追加内容
$ echo hello >> index.txt

# cat 查看文件内容

$ cat 文件名

# cd切换目录

$ cd 目录路径
  • 切换到上级目录
$ cd ..

# ls 查看当目录下有哪些文件

$ ls
  • al 查看所有文件及文件信息
$ ls -al

# pwd查看当前所在目录路径

$ pwd

# rm 递归删除

$ rm -rf 文件名

# cp复制文件

$ cp 1.js 2.js

-r递归复制

$ cp -r dir/ dir2

# mv移动文件

$ mv 1.js a/1.js

# clear清屏

$ clear

# vim常用命令

在命令终端输入vim, 进入vim编模式

$ vim

当 git commit 没有输入提交信息时 也会进入vim编辑模式

默认是命令模式

a/i/o 进入输入模式

esc 退出输入模式

: 进入末行模式

命令模式下 输入的命令

:wq 保存并退出
:q! 强制退出
:q 退出不保存
:w 保存但不退出
