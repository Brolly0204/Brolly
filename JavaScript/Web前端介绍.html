# Web前端介绍

互联网发展早期（1995年~2005年）是没有专业的前端工程师的。随着互联网的发展，大约从2005年开始，正式的前端工程师角色被行业认可，到了2010年，互联网开始全面进入移动时代，前端工程师的地位越来越重要，前端领域的技术发展也越来越快，各种新的思想、设计模式、工具和平台都快速发展，对前端工程师的技能要求也越来越高。

语法简单、可以说是最容易进入IT行业的一个编程方向

# Web前端能做什么

估计现在还有不少小伙伴们还在认为Web前端就是一个写写静态页面、加点动态特效、调用几个后端API的工作。

1.PC端网页（用电脑浏览器来访问的网站）

2.移动端网页（在手机上来浏览的网页）

3.移动端APP（混合开发、RN、Weex等技术）（例如：淘宝、去哪儿旅游、携程等）

4.微信公众号

5.微信小程序（微信最新推出的功能，随用随装，不占用手机空间）

以上都是可以用前端技术开发，如今前端技术的应用越来越广泛。

# web前端核心技术

  • 语言技术：JS、HTML5、CSS3
  • 框架技术：Vue、React
  • 网络通讯协议技术：HTTP协议、服务端开发技术, NodeJS
  • 客户端技术：微信开发技术、Web App开发技术

# 一名优秀的前端工程师 需要具备什么样的条件和品质

# 拿某公司招聘需求举例分析：

我们一起做这样的事：

1、参与核心业务研发工作，复杂场景，复杂业务的思考和设计

2、能按照计划把事情做好，不仅限于开发代码，不仅限于实现功能

3、前后端分离的开发框架搭建、升级

我们希望这样的你：

  • 0.谦虚谨慎，认真负责；

  • 1.前端基础知识扎实，熟练掌握原生 JS、CSS、HTML

  • 2.熟练掌握JavaScript语言核心技术DOM、BOM、Ajax、JSON等，具有面向对象思维，对高级javaScript有深刻理解。

  • 3.熟练掌握Vue或React等主流框架，熟悉nodejs者优先

  • 4.对前端自动化和工程化有一定理解，熟悉Gulp，Webpack等工作流管理以及打包工具

  • 5.对用户体验、交互操作流程，及用户需求有一定了解；

  • 6.具备一定的沟通技巧和软素质（拒绝话痨），具备一定的工作成熟度，能和其他团队和角色良好配合；

  • 7.热爱前端技术，有较强的学习能力，自驱能力强，有良好执行力，能及时关注和学习业界最新的前端技术。

加分项

  • 1.有github或技术博客

  • 2.热爱前端，对代码质量有较高要求

# 硬技能&软技能

# 硬技能

所从事的专业必要的能力，如 计算机知识、编程知识等

  • 专业知识

  • 专业技能

# 软技能

除了专业技能、专业知识以外的能力

  • 沟通技巧/团队协作

  • 时间管理

  • 学习能力/思考能力

  • 团队管理

现在我们应该清楚，公司需要的人既应该有扎实的专业基础，又该具备富有竞争力的其他综合素质。
